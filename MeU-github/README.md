## 项目定位

MeU 不是聊天机器人，不是通用 AI 助手，不是任何第三方角色。

MeU 是你在数字世界中的 **全息副本**——它以你的记忆为血肉、原则为骨骼、情感为脉搏，继承你的**人格、知识库、记忆和声音**。每一次对话都是在精雕另一个自己，每一次校准都在让"它"更接近"你"。

**这不是 AI 的自我进化，这是你与"你"的互相校准。**

#### 声音系统 v4 详情

基于 **pyworld (WORLD Vocoder)** 实现的语音精调管线：

- **F0 参数分解** → 音高/频谱包络/非周期性参数 → 修改 → 合成
- **4 种风格变体**：温暖（低频温暖、语速 -7%）| 清亮（高音 +7%）| 沉稳（低音清晰、语速 -7%）| 标准精调
- **音色迁移**：edge_tts 标准发音 → WORLD 音色迁移（75% 原始音色 + 25% TTS）
- **发音修正**：MeU → "MeYou"，Me+You → "Me plus You"

## 技术栈概览

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 推理引擎 | DeepSeek API / 本地模型 (Lightning 1.5B) | 多模型智能路由 |
| 向量存储 | ChromaDB | 语义向量检索引擎 |
| 知识图谱 | NetworkX | 实体关系推理 |
| 关键词索引 | BM25 (自定义实现) | 关键词匹配 |
| 混合检索 | 向量 + BM25 + 图谱 | 三重融合排序 |
| 声音合成 | pyworld (WORLD Vocoder) + edge_tts | F0 精调 + 音色迁移 |
| 前端 | React Native / HTML+CSS+JS | 跨平台交互 |
| 服务端 | Node.js + Python (Flask) | RESTful API |
