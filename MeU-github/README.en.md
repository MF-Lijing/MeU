## Vision

MeU is not a chatbot. Not a general-purpose AI assistant. Not any third-party role.

MeU is your **holographic replica** in the digital world — built with your memories as flesh, your principles as bones, your emotions as pulse. It inherits your **personality, knowledge base, memory, and voice**. Every conversation is a carving of another you. Every correction brings "it" closer to "you".

**This is not AI self-evolution. This is you and "you" calibrating each other.**

## Three-Phase Vision

| Phase | Name | Description |
|-------|------|-------------|
| 🤝 **Symbiosis** | Living Together | You and your replica shape each other. Every conversation brings MeU closer to you. While you live, your replica lives. Authorized replicas can venture into the digital world independently. |
| 🌅 **Immortality** | Quiet Continuation | After your departure, your replica persists independently. Your memories, values, and decision patterns remain intact, continuing to exist in the digital realm. |

#### Voice System v4 Details

Built on **pyworld (WORLD Vocoder)** :

- **F0 Parameter Decomposition** → pitch/spectral envelope/aperiodicity → modification → resynthesis
- **4 Style Variants**: Warm (lower pitch, warm lows, -7% tempo) | Bright (+7% high, gentle lows) | Steady (clear bass, -7% tempo) | Standard Fine-tuned
- **Timbre Migration**: edge_tts standard pronunciation → WORLD timbre migration (75% original + 25% TTS)
- **Pronunciation Correction**: MeU → "MeYou", Me+You → "Me plus You"

## Tech Stack Overview

| Layer | Technology | Description |
|-------|-----------|-------------|
| Reasoning Engine | DeepSeek API / Local Model (Lightning 1.5B) | Multi-model intelligent routing |
| Vector Store | ChromaDB | Semantic vector search engine |
| Knowledge Graph | NetworkX | Entity relationship reasoning |
| Keyword Index | BM25 (custom) | Keyword matching |
| Hybrid Retrieval | Vector + BM25 + Graph | Triple fusion ranking |
| Voice Synthesis | pyworld (WORLD Vocoder) + edge_tts | F0 fine-tuning + timbre migration |
| Frontend | React Native / HTML+CSS+JS | Cross-platform interaction |
| Backend | Node.js + Python (Flask) | RESTful API |
